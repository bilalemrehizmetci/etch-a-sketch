    
# Etch-a-Sketch

A simple web-based Etch-a-Sketch application built with HTML, CSS, and JavaScript. Draw on a grid by hovering your mouse over the squares. You can change the grid size, select different colors, use a random color mode, and reset the board.

## Features
- Adjustable grid size (2x2 to 100x100)
- Draw by hovering over squares
- Color selection (black, random, or custom)
- Reset board to clear drawing
- Toggle drawing mode (active/inactive)

## How to Use
1. **Open `index.html` in your browser.**
2. **Set the grid size:** Enter a number between 2 and 100 and submit to change the board size.
3. **Choose a color:** Select black, random, or another color (if implemented).
4. **Draw:** Hover your mouse over the grid squares to draw.
5. **Reset:** Use the reset button to clear the board.
6. **Toggle mode:** Click anywhere on the page to toggle between drawing (active) and not drawing (inactive).

## Files
- `index.html` – Main HTML file
- `styles.css` – Styles for the app
- `reset.css` – CSS reset
- `main.js` – JavaScript logic for the Etch-a-Sketch

## Customization
You can modify the code to add more colors, change the default grid size, or enhance the UI.
